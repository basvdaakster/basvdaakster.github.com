NuggetEco

NuggetEco is an economy plugin for bukkit. NuggetEco is currently in development.

NuggetEco is not a standard economy plugin. Most economy plugins use a "virtual" balance, NuggetEco does not. NuggetEco uses Gold Nuggets as coins, you must have Gold Nuggets in your inventory to buy items, so think twice before heading out to a shop in a PvP world. You must keep your coins safe yourself, there is no bank, I recommend using a protection plugin so you can create a secure chest to put your Nuggets in.

Installation / Configuration

  • Download nuggeteco.zip and extract the contents in to bukkit's plugin folder.
  • Configure the prices.yml to your likings or use the default prices.yml file.
  • Enjoy
